Sidebars

  • Sidebars can be left or right.
  • Columns class/size should be col-lg-3 col-md-3 col-sm-3.
  • Margins should match element it's next to, ideally 40px 0.
  • Padding should be 30px.
  • $lightblue background.
  • Border-radius of 4px.
